## 📊 Features
- Clean and professional dashboard layout
- Charts for sales analysis
- Monthly performance tracking
- Easy-to-read visual reports
- Dynamic data representation

This dashboard helps businesses and individuals analyze sales trends, monitor performance, and make better decisions.
